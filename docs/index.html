‚ö†Ô∏è Important Information About IP Locks

  • Free, Donator, and Contributor tiers have IP-locked keys (1 IP per key)
  • Subscriber tiers and above get IP-free keys
  • To reset your IP, use /user resetip or the Reset IP button in the control panel
  • After resetting, your next API request sets the new IP
  • Automating IP resets is forbidden and may result in a ban

üíª Code Examples

Python Example

# Install the OpenAI package: pip install openai
import openai

client = openai.OpenAI(
    base_url="https://api.crusont.com/v1",
    api_key="your-api-key-here",
)

response = client.chat.completions.create(
    model="gpt-4o-mini",  # or claude-3-haiku, gemini-1.5-flash, etc...
    messages=[{"role": "user", "content": "Hello, AI!"}],
)

print(response.choices[0].message.content)

JavaScript Example

// Install the OpenAI package: npm install openai
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://api.crusont.com/v1',
  apiKey: 'your-api-key-here',
});

async function main() {
  const response = await client.chat.completions.create({
    model: 'gpt-4o-mini', // or claude-3-haiku, gemini-1.5-flash, etc...
    messages: [{ role: 'user', content: 'Hello, AI!' }],
  });

  console.log(response.choices[0].message.content);
}

main();

cURL Example

curl -X POST "https://api.crusont.com/v1/chat/completions" \
  -H "Authorization: Bearer your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-mini",
    "messages": [{"role": "user", "content": "Hello, AI!"}]
  }'

Token System Overview

Daily Token Grants

Every 24 hours, you are eligible for a token grant if your current token balance is below 5000 tokens. You can check both of these values anytime on the Discord Server with /user lookup.

Token Equivalence

Tokens are equivalent across all models and endpoints. 1 token = 1 token, regardless of the model you're using.

‚ùì Frequently Asked Questions

API Questions

How do I get my API key?

Join our Discord server and use /user get-key or visit our web dashboard to create multiple keys.

How does the IP lock system work?

Free tier accounts are limited to one IP address for security. Premium tiers get IP-free access. You can reset your IP with /user resetip.

Where can I use my API key?

Anywhere! Our API is compatible with OpenAI's format, so you can use it with any OpenAI-compatible client or library.

How do I troubleshoot API errors?

Check your API key, ensure you have sufficient credits, and verify the model is available for your tier. Join our Discord for support.
